# Library

A small Library web app. Allows users to add, view, update, and remove books.This project is part of the [JavaScript Course](https://www.theodinproject.com/lessons/node-path-javascript-library) from [The Odin Project](https://www.theodinproject.com).

## Features

- Add books via form (title, author, number of pages, read status)
- Display all books using DOM manipulation
- Change read status for each book
- Remove books from the library

## Built With

- HTML
- CSS
- JavaScript
